# Gerador de Evoluções Clínicas — Acolher e Cuidar

Site simples para gerar evoluções terapêuticas rápidas.

## Arquivos

- `index.html`: página principal
- `css/style.css`: visual do site
- `js/script.js`: funcionamento do gerador
- `assets/banner-clinica.png`: imagem usada no topo do site

## Como testar no computador

1. Abra a pasta.
2. Clique duas vezes em `index.html`.
3. O site abrirá no navegador.

## Como trocar a imagem

Substitua o arquivo:

`assets/banner-clinica.png`

por outra imagem com o mesmo nome.

## Como colocar online grátis na Vercel

1. Crie uma conta em vercel.com.
2. Crie uma conta em github.com.
3. Suba esta pasta em um repositório no GitHub.
4. Na Vercel, clique em “Add New Project”.
5. Selecione o repositório.
6. Clique em “Deploy”.
7. A Vercel vai gerar um link gratuito, por exemplo:

`gerador-evolucoes-acolher-cuidar.vercel.app`

## Observação

Este site não salva dados de pacientes. Ele apenas gera o texto na tela e permite copiar.
Isso é melhor para começar, porque evita misturar com o sistema oficial da clínica.
